## 本地变量的规则

    本地变量定义的块内的
    
      它可以是定义在函数的块内
      
      也可以定义在语句的块内
      
      甚至可以随便拉一堆大括号来定义变量
      
    程序运行进入这个块之前，其中的变量不存在，离开这个块，其中的变量就消失了
    
    块外面定义的变量在里面仍然有效
    
    块里面定义了盒外面同名的变量则掩盖了外面的
    
    不能在一个块内定义同名的变量
    
    本地变量不会被默认初始化

### 数组的大小

    sizeof给出整个数组所占据的内容的大小，单位是字节
    
    sizeof(a)/sizeof(a[0])
    
    sizeof(a[0])给出数组中单个元素的大小，于是相除就得到了数组的单元个数
    
    这样的代码，一旦修改数组中初始的数据，不需要修改便利的代码
